### elephant-mobile

```
yarn dev // 启动项目

```

基于 react antd-mobile redux react-redux webpack 等的最新配置。

实现的功能：

-   支持 Ant-Design-Mobile 按需加载
-   支持 babel-polyfill 按需加载
-   支持预请求
-   tree shaking
-   PWA
-   CSS 模块化
-   小图片的 base64 处理
-   React 懒加载，按需加载，代码分割
-   支持 less 预处理
-   Code Splitting
-   提取公共代码，打包成一个 chunk
-   CSS 压缩
-   增加 CSS 前缀，兼容各种浏览器
-   对于各种不同文件打包输出指定文件夹下
-   将 CSS 文件单独抽取出来
